Política de Privacidade

Última atualização: 03/12/2025

1. Coleta de Dados

Coletamos apenas as informações que você fornece voluntariamente, como nome, e-mail e conteúdo das homenagens.

2. Uso das Informações

Seus dados são utilizados exclusivamente para melhorar a experiência dentro do aplicativo, garantindo funcionalidades personalizadas.

3. Compartilhamento de Dados

Não compartilhamos suas informações com terceiros sem o seu consentimento explícito.

4. Segurança

Aplicamos medidas de segurança modernas para proteger seus dados contra acesso não autorizado.

5. Consentimento

Ao usar este aplicativo, você concorda com nossa Política de Privacidade.
